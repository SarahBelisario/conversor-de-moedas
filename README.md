## 📌 Sobre o Projeto

O **Conversor de Moedas** é uma aplicação web desenvolvida para realizar a conversão rápida e intuitiva de diferentes moedas estrangeiras (**Dólar**, **Euro** e **Libra**) para o **Real Brasileiro (BRL)**. 

A aplicação conta com validação de dados em tempo real, formatação de valores no padrão de moeda local e exibição dinâmica do resultado.

---

## 🚀 Funcionalidades

- [x] **Seleção de Moedas:** Suporte para conversão de Dólar Americano (USD), Euro (EUR) e Libra Esterlina (GBP).
- [x] **Validação de Entrada:** Restrição no campo de valor para permitir apenas números via Regex, prevenindo erros de digitação.
- [x] **Formatação Monetária:** Utilização da API nativa `Intl.NumberFormat` (`toLocaleString`) para formatar os valores no padrão de moeda brasileira (R$).
- [x] **Exibição Dinâmica:** Apresentação clara da taxa de câmbio utilizada e do valor final convertido ao enviar o formulário.
- [x] **Tratamento de Erros:** Bloco `try...catch` para captura e tratamento de possíveis exceções no processo de conversão.
